# Dólar Blue Hoy 

Este proyecto es una aplicación web simple que permite consultar la cotización actual del dólar blue en Argentina. Utiliza una API pública para obtener los datos en tiempo real y los muestra de forma clara al usuario.

 Funcionalidades
- Consulta del precio de compra y venta del dólar blue
- Actualización manual con un botón
- Interfaz sencilla y directa

 Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- API: [DólarSi](https://www.dolarsi.com/api/api.php?type=valoresprincipales)
- Hosting: Vercel
